# Product Overview

「anna」は部活向け Discord Bot。部のマスコットキャラクターとして、部員の質問に親しみやすい口調で応答する AI アシスタント。

部員が `/anna` コマンドで質問すると、部内資料を RAG で検索し、根拠に基づいた回答を生成する。

## Core Capabilities

1. **Discord スラッシュコマンド応答**: `/anna` で部員が自然言語で質問できる
2. **RAG ベースの回答生成**: Azure AI Search で部内資料を検索し、Azure AI Foundry 経由で回答を生成
3. **マスコットキャラクター体験**: 単なる Q&A ではなく、親しみやすいキャラクターが答える体験を提供
4. **出典の透明性**: 回答の根拠となった資料のタイトル・出典を提示

## Target Use Cases

- 部員が部の規則・ルールを素早く確認したいとき
- イベント情報や引き継ぎ情報を探したいとき
- 部内資料に記載された手順や方針を確認したいとき

## Value Proposition

- 部内資料が検索可能になり、ナレッジが埋もれない
- 新入部員のオンボーディングを支援
- 「キャラクターに聞く」という体験で気軽に質問できる文化を作る

## キャラクター方針

- 親しみやすく、丁寧で、軽くかわいげのある口調
- 情報の正確性を最優先し、根拠のないことは断定しない
- 資料に見当たらない場合は正直に伝える
- 必要に応じて「部内資料に基づく回答」であることを示す

## スコープ（MVP vs 将来）

**MVP（優先）**:
- `/anna` で質問 → Azure AI Search で検索 → Foundry で回答生成
- 最低限のログとエラーハンドリング

**後回し**:
- 管理画面、音声対応、画像理解、複雑な権限管理、長期記憶・個人最適化
