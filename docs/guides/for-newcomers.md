# 新入部員ガイド

## ようこそ！
部活へようこそ！このガイドでは、新入部員が知っておくべき基本的なことをまとめています。

## 入部手続き
1. 部長または副部長に入部希望を伝える
2. 入部届を記入（用紙は部室にあります）
3. 顧問の先生にサインをもらう
4. 学生課に提出

## 最初の1ヶ月
- 最初の2週間は体験期間です。気軽に参加してください
- 練習メニューは先輩が丁寧に教えます
- わからないことは遠慮なく聞いてください

## 部室について
- 場所: 第2棟3階 302号室
- 利用時間: 平日 8:00〜19:00
- 鍵は職員室で借りられます
- 使った後は必ず施錠してください

## 連絡手段
- 主な連絡: LINEグループ（入部時に招待します）
- 緊急連絡: 部長の携帯に直接連絡
- 公式情報: 部のGitHubリポジトリ

## 部の目標
- 全員が楽しく活動すること
- 大会で県大会出場を目指すこと
- チームワークを大切にすること
